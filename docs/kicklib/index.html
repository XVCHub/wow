kick library

booting the library

load the library using the following code snippet

lua 
loadstring(game:HttpGet("https://egirlswow.pages.dev/libs/kick/source.lua"))()

configure the settings

customize the kick screen appearance and behavior

lua 
getgenv().Config = {
    title = "Disconnected",
    message = "You have been kicked from this experience.",
    type = "1", -- types: 1 = only leave button, 2 = leave and reconnect buttons
    button1 = "Leave",
    button2 = "Reconnect"
}
configuration options
type "1": displays only the leave button
type "2": displays both leave and reconnect buttons

button callbacks

define what happens when buttons are clicked

lua 
getgenv().Callbacks = {
    button1_callback = function()
        game:Shutdown() -- leave
    end,
    button2_callback = function()
        game:GetService("TeleportService"):Teleport(game.PlaceId) -- reconnect
    end
}
